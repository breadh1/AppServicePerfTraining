# Contributing to Azure App Service Performance Training

Thank you for your interest in contributing to this training repository! We welcome contributions from the community.

## Ways to Contribute

- **Fix typos or improve documentation**: Submit a pull request
- **Add new examples**: Share your performance optimization examples
- **Report issues**: Open an issue for bugs or unclear content
- **Suggest improvements**: Propose new training modules or labs
- **Share experiences**: Add case studies or real-world scenarios

## Getting Started

1. **Fork the repository**
2. **Create a branch** for your changes
3. **Make your changes** following our guidelines
4. **Test your changes** (if applicable)
5. **Submit a pull request**

## Guidelines

### Documentation

- Use clear, concise language
- Include code examples where appropriate
- Follow markdown best practices
- Add links to official Microsoft documentation
- Test all code samples

### Code Samples

- Use latest stable versions of frameworks
- Include comments explaining key concepts
- Follow language-specific best practices
- Include error handling
- Make samples self-contained when possible

### Lab Exercises

- Clearly state objectives and prerequisites
- Provide step-by-step instructions
- Include verification steps
- Add troubleshooting section
- Estimate completion time

## Code of Conduct

- Be respectful and inclusive
- Welcome newcomers
- Accept constructive criticism
- Focus on what's best for the community

## Questions?

Open an issue or reach out to the maintainers.
